# École Polytechnique  ~ INF582 Data Challenge 

## Overview
This repository contains the code and resources for a NLP project focused on text summarization using fine-tuned language models. The goal of the project is to to develop and implement generative methods to get automatic summaries and headlines for huge articles.

## Installation 📥
1. Ensure you have Python 3.x (with pip) before running the project

2. Clone the repository:
   ```bash
   git clone https://github.com/czartur/abstractive_summarization.git
   cd abstractive_summarization

3. Install the required dependencies:
    ```bash
    pip install -r requirements.txt

## Usage🚦
To use the project, follow these steps:

1. Prepare your dataset or use the provided example dataset.
2. Choose your fine-tuned model.
3. Train the model using the provided scripts.
4. Evaluate the model on test data.
